# ST-004: Pages d'authentification Frontend

**Titre:** UI Login/Register avec NextAuth.js ou custom forms

**User Story:** En tant que développeur frontend, je veux créer les pages auth pour l'UX utilisateur.

**Critères d'acceptation:**
- [ ] /login et /register responsive avec shadcn/ui
- [ ] Form validation (Zod)
- [ ] Appel API + stockage JWT (localStorage/cookies)
- [ ] Redirect /dashboard post-login

**Tâches techniques:**
- Installer @hookform/resolvers/zod lucide-react
- Composants: Card, Input, Button, Form
- API utils avec fetch/axios
- Protected routes avec middleware
- Loading/error states

**Dépendances:** ST-001, ST-003

**Priority:** Must
